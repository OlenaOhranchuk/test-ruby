1. Install Node.js:
	sudo apt update
	sudo apt install nodejs

2. Go the project's root.

3. Set the git hooks folder:
	git config core.hooksPath hooks

4. Install bundler:
	gem install bundler

5. Install gems:
	bundle exec bundle install --path vendor/bundle

6. Create, migrate and seed the database:
	bundle exec rake db:create
	bundle exec rake db:migrate
	bundle exec rake db:seed

7. Run the server:
	bundle exec rails s
