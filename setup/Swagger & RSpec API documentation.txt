1. Download Swagger UI at https://github.com/swagger-api/swagger-ui.

2. Copy "dist" folder from Swagger UI project to "publinc/swagger/" in the project.

3. Add this route in your "config/routes.rb" file:
  get 'api' => redirect('/swagger/dist/index.html?url=/api/open_api.json')

4. Install RSpec API documentation gem:
  https://github.com/zipmark/rspec_api_documentation

5. Set these configuration options in you spec/spec_helper.rb file:
  RspecApiDocumentation.configure do |config|
    config.docs_dir = Rails.root.join("public", "api")
    config.format = :open_api
    config.request_body_formatter = Proc.new { |params| params.to_json }
    config.client_method = :rspec_api_client
  end

6. Add a few tests in spec/acceptance folder.

7. Generate JSON file for Swagger:
  bundle exec rake docs:generate

8. Check if swagger works correctly by visiting localhost:3000/api.
